# Legal Document Intelligence System

This project builds an NLP pipeline for civil rights legal case analysis using the Multi-LexSum dataset.

## Version 1 Goal

Core components:

- Load and explore the Multi-LexSum dataset
- Clean long legal case documents
- Generate long, short, and tiny summaries
- Train classification models for class action prediction
- Train classification models for case type prediction
- Evaluate summarization and classification performance
- Build an interactive Streamlit demo
